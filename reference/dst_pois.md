# Poisson Distribution

Makes a Poisson distribution.

## Usage

``` r
dst_pois(lambda)
```

## Arguments

- lambda:

  Mean of the Poisson distribution; single positive numeric.

## Value

A Poisson distribution.

## Examples

``` r
dst_pois(1)
#> Poisson distribution (discrete) 
#> --Parameters--
#> lambda 
#>      1 
```
