# F Distribution

Makes an F distribution.

## Usage

``` r
dst_f(df1, df2)
```

## Arguments

- df1, df2:

  Degrees of freedom of the numerator and denominator, both single
  positive numerics.

## Value

An F distribution.

## Examples

``` r
dst_f(2, 3)
#> F distribution (continuous) 
#> --Parameters--
#> df1 df2 
#>   2   3 
```
