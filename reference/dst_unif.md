# Uniform Distribution

Makes a Uniform distribution.

## Usage

``` r
dst_unif(min, max)
```

## Arguments

- min, max:

  Minimum and maximum of the distribution. Single numerics.

## Value

A Uniform distribution.

## Examples

``` r
dst_unif(0, 1)
#> Uniform distribution (continuous) 
#> --Parameters--
#> min max 
#>   0   1 
```
