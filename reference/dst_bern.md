# Bernoulli Distribution

Makes a Bernoulli distribution, representing the outcome of a single
trial with a given success probability.

## Usage

``` r
dst_bern(prob)
```

## Arguments

- prob:

  Probability of success; single numeric between 0 and 1.

## Value

A Bernoulli distribution.

## Examples

``` r
dst_bern(0.3)
#> Bernoulli distribution (discrete) 
#> --Parameters--
#> prob 
#>  0.3 
```
