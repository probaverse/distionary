# Gamma Distribution

Makes a Gamma distribution.

## Usage

``` r
dst_gamma(shape, rate)
```

## Arguments

- shape:

  Shape parameter; single positive numeric.

- rate:

  Rate parameter; single positive numeric.

## Value

A Gamma distribution.

## Examples

``` r
dst_gamma(2, 1)
#> Gamma distribution (continuous) 
#> --Parameters--
#> shape  rate 
#>     2     1 
```
