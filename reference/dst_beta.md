# Beta Distribution

Makes a Beta distribution.

## Usage

``` r
dst_beta(shape1, shape2)
```

## Arguments

- shape1, shape2:

  Shape parameters of the distribution; single positive numerics.

## Value

A Beta distribution.

## Examples

``` r
dst_beta(2, 3)
#> Beta distribution (continuous) 
#> --Parameters--
#> shape1 shape2 
#>      2      3 
```
